# Hello Lab

Тестовый JS-проект в монорепозитории: Next.js (App Router), Tailwind CSS с префиксом `hl-` для изоляции классов.

## Команды

Из корня репозитория:

- `npm run dev:hello-lab` — разработка на [http://localhost:3001](http://localhost:3001)

В этой папке: `npm run dev`, `npm run build`, `npm start`.

## Регистрация

Файл `project.manifest.json` подхватывается порталом (`apps/portal`) и показывается в списке проектов.

## Docker

Из корня: `docker compose up --build hello-lab`.
