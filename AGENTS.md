# Инструкции для агента в этом репозитории

## Маркер завершения (Telegram relay)

Если используется relay `projects/js/cursor-agent-telegram` и в его `.env` задан `TELEGRAM_DONE_PHRASE`, **каждый** ответ ассистента пользователю в чате Cursor должен заканчиваться **отдельной последней строкой**, дословно совпадающей с переменной (по умолчанию для монорепозитория):

```text
<<<CURSOR_RELAY_DONE>>>
```

Без этой строки релей не считает ответ завершённым для уведомления «готово» в Telegram.

Правило не отменяется для коротких ответов («привет», «ок»). Маркер не помещать в блоки кода (\`\`\`).

Подробнее: [.cursor/rules/relay-completion-marker.mdc](.cursor/rules/relay-completion-marker.mdc).
