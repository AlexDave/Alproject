# Папка `agents`

Здесь лежат **справочные материалы для настройки AI-агентов** (Cursor, Claude Code и аналоги). Это **не** часть рантайма портала и микропроектов в `projects/` — в приложение этот код не подключается.

## Содержимое

| Путь | Что это |
|------|---------|
| `everything-claude-code/` | Вендорный репозиторий [Everything Claude Code](https://github.com/affaan-m/everything-claude-code): готовые `.cursor/rules`, skills, hooks, команды, примеры MCP, гайды по памяти сессий, безопасности и оптимизации токенов. |

## Когда Cursor-агенту сюда заходить

Обращайся к `agents/everything-claude-code/` (читай файлы, копируй или адаптируй паттерны), если задача про:

1. **Правила и стиль под агентов** — project rules, соглашения по TS/Python/Go и общие `common-*` правила.
2. **Skills и команды** — оформление `SKILL.md`, slash-команды, субагенты.
3. **Hooks** — события сессии, до/после чтения файла, shell, MCP и т.д.
4. **MCP** — примеры конфигов и серверов для tool-calling.
5. **Память и контекст** — персистентность между сессиями, компактизация, «continuous learning» из сессий.
6. **Безопасность агентов** — sandbox, санитизация, риски prompt injection / tool abuse (см. security-гайды в репо).
7. **Параллельные агенты** — worktrees, каскады, оценка качества ответов (evals).

Если правишь только бизнес-логику портала или микросервис в `projects/` и **не** трогаешь обвязку агентов — эту папку можно не открывать.

## Расположение

`agents/` стоит **рядом с** `projects/` в корне монорепозитория Alproject.

## Уменьшить размер клона (опционально)

Каталог `everything-claude-code/` большой и может содержать вложенный `.git`. Чтобы не держать его в основном дереве:

1. Удалите локально `agents/everything-claude-code` (или не коммитьте его в свой форк).
2. Подключите апстрим как submodule:  
   `git submodule add https://github.com/affaan-m/everything-claude-code.git agents/everything-claude-code`  
   (после инициализации репозитория в корне Alproject).
3. Либо клонируйте ECC отдельно и открывайте файлы по ссылке из таблицы выше — для Cursor достаточно читать копию вне монорепо.
